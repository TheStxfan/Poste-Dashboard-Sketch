# Obiettivo Progetto: Dashboard Servizio Poste

L'obiettivo è la creazione di una dashboard (HTML, CSS, JS) professionale rivolta a un **Responsabile IT** per il monitoraggio di un servizio postale (ispirato al modello di Poste Italiane).

## Requisiti della Dashboard
La dashboard deve includere le seguenti sezioni e funzionalità:

- **Stato delle code in tempo reale**: Monitoraggio dell'afflusso e della gestione degli utenti agli sportelli.
- **KPI principali**: Indicatori chiave di prestazione del servizio.
- **Carico server**: Monitoraggio dell'infrastruttura tecnologica sottostante.
- **Tempo medio di risposta**: Performance dei sistemi e dei servizi al cittadino.
- **Eventuali alert critiche**: Sistema di notifiche per anomalie o interruzioni di servizio.

## Note Tecniche
- **Target**: IT Manager.
- **Tecnologie**: HTML5, CSS3, JavaScript.
- **Dati**: I dati dovranno essere estratti e integrati a partire dal file Excel fornito successivamente.
